Un widget con tus sitios sociales: Facebook, Twitter, Google Plus y FeedBurner al estilo FW

== Description ==

Todos tus sitios web en un solo widget: Facebook, Twitter, Google Plus y FeedBurner.

Haz que tus visitantes se suscriban y se mantengan conectados todo el tiempo, un widget de facil acceso que solo requiere que llenes tus datos.

== Installation ==

1. Sube al archivo .zip a través de tu pantalla de administración de WordPress o sube la carpeta por FTP.
2  Activa el plugin en el menu Plugins
3. Ve a la página de Widgets y arrastralo para activarlo.
4. Rellena los datos de tu sitio web.
5. Disfruta.
